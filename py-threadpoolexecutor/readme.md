# CS 3820: ThreadPoolExecutor Lab Exercises
## Submission Instructions

---

## Overview

This lab consists of 5 exercises designed to teach you ThreadPoolExecutor fundamentals through hands-on practice. Each exercise builds on the previous one, progressing from basic concepts to building a complete concurrent application.

---

## Submission Requirements

### Git Repository Structure

All submitted assignments must be checked into the **YU GitHub system** into the **Parallel branch** that you created for this course.

**This assignment will be checked into the `/lab-py-threadpool` folder.**

### Required File Structure

```

/lab-py-threadpool/
    ├── lab1.py                (Exercise 1: Your First ThreadPoolExecutor)
    ├── lab2.py                (Exercise 2: map() vs submit())
    ├── lab3.py                (Exercise 3: Exception Handling)
    ├── lab4.py                (Exercise 4: Progress Tracking)
    └── lab5.py                (Exercise 5: Web Scraper)
```

**All 5 `.py` files must be in the `/lab-py-threadpool` directory.**
